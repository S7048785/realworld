# 文章标题下划线动画实现

## 效果展示

当鼠标悬浮在文章标题上时，标题下方会出现一条从右到左延伸的下划线。

## 代码实现

### CSS 样式 (`ArticleItem.css`)

```css
.title {
	/* 下划线使用渐变色，初始宽度为 0 */
	background: linear-gradient(to right, var(--color-primary), var(--color-primary)) no-repeat right bottom;
	background-size: 0 1px;
	transition: background-size 0.3s ease-in-out;
}

.title:hover {
	/* 悬浮时：背景位置从左侧开始，宽度为 100% */
	background-position: left bottom;
	background-size: 100% 1px;
}
```

### HTML 结构

```tsx
<Link to={"/article/" + article.id} className="text-lg title">
	{article.title}
</Link>
```

## 实现原理

### 1. `background` 多重属性

```css
background:

linear-gradient
(
to right,

var
(
--color-primary

)
,
var
(
--color-primary

)
)
no-repeat
right bottom

;
```

这里巧妙地将 `background` 用于绘制下划线：

- `linear-gradient(...)` - 创建一个单色渐变（实际是纯色）
- `no-repeat` - 不重复
- `right bottom` - 初始位置在**右下角**

### 2. `background-size: 0 1px`

- **宽度 0** - 下划线初始不可见
- **高度 1px** - 下划线厚度

### 3. 悬浮状态切换

```css
.title:hover {
	background-position: left bottom; /* 位置移到左下角 */
	background-size: 100% 1px; /* 宽度变为 100% */
}
```

**动画过程**：

1. 初始：背景在右下角，宽度为 0
2. 悬浮：背景位置移到左下角，宽度从 0 过渡到 100%
3. `transition` 产生平滑过渡效果

---

## 为什么从右到左？

通过控制 `background-position`：

- **初始状态**：`right bottom` + `background-size: 0` → 不可见
- **悬浮状态**：`left bottom` + `background-size: 100%` → 从左向右展开

实际上，因为初始位置在右侧且宽度为 0，当宽度增加到 100% 时，会向左延伸形成从右到左的效果。

---

## 扩展：从左到右动画

如果需要从左到右的动画效果：

```css
.title {
	background: linear-gradient(to right, var(--color-primary), var(--color-primary)) no-repeat left bottom;
	background-size: 0 1px;
	transition: background-size 0.3s ease-in-out;
}

.title:hover {
	background-position: left bottom;
	background-size: 100% 1px;
}
```

关键：初始 `background-position` 放在 `left bottom`。

---

## 扩展：虚线下划线

```css
.title {
	background: linear-gradient(to right, var(--color-primary) 50%, transparent 50%) no-repeat left bottom;
	background-size: 8px 2px;
	transition: background-size 0.3s ease-in-out;
}

.title:hover {
	background-size: 100% 2px;
}
```

## 优点

| 特性        | 说明                       |
|-----------|--------------------------|
| **纯 CSS** | 无需 JavaScript            |
| **性能好**   | 使用 GPU 加速的 background 属性 |
| **灵活**    | 可轻松修改颜色、粗细、动画速度          |
| **响应式**   | 自动适应不同长度的文本              |
